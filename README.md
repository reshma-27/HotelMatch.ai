# HotelMatch AI Prototype

A Streamlit prototype for a prompt-based hotel booking assistant.

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy on Streamlit Community Cloud
1. Create a public GitHub repository.
2. Upload `app.py`, `requirements.txt`, and `README.md`.
3. Sign in at [Streamlit Community Cloud](https://streamlit.io/cloud).
4. Click **Create app** and connect your GitHub repository.
5. Select `app.py` as the main file and deploy.

Streamlit Community Cloud supports free deployment for public apps.
